**8.سایر بانک‌های اطلاعاتی:** استفاده از این قابلیت، این امکان را به کاربر خواهد داد تا برای ساخت گزارش مدنظر اطلاعات موردنیاز خود از سایر بانک‌های اطلاعاتی دیگر به منابع این گزارش اضافه کند.

 در صورتی که از گزینه **سایر بانک‌های اطلاعاتی** استفاده کردید باید اطلاعات زیر را پر کنید . 
 
 ![سایر بانک های اطلاعاتی](./Images/report7.png)
 
 **نام:**  یک نام برای این منبع بانک اطلاعاتی انتخاب کنید.
 
 **آدرس سرور:** در این بخش آدرس سرور بانک اطلاعاتی را وارد کنید.
 
 **نوع دسترسی:**  دسترسی به دیتابیس موردنظر از دو طریق امکان‌پذیر است:
 
 **1. Windows Authentication:**      از طریق user که لاگین است.
 
 **2. SQL Server Authentication:** از طریق user و پسورد مشخص دیتابیس.
